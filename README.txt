webstats é um site de estatisticas de jogos onde os players guardam e comparam os seus melhores resultados para ganharem "prémios"
cada utilizador pode ter uma conta a que pode associar os jogos que possui e enviar os seus resultados do jogo para o site
alem da homepage existe uma pagina de users em que se pode dar inspect a cada utlizador e listar os seus jogos e ultimos scores adicionados
existe uma pagina games, em que aparece uma lista de todos os jogos disponíveis (e caso o user esteja authenticado pode associar o jogo à sua conta)
existe uma pagina de highscores que mostra os highscores para cada jogo
há tambem uma pagina de perfil em que cada user tem as infos da sua conta
